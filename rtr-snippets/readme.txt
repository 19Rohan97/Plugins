A simple plugin that allows users to add custom CSS and JavaScript to their WordPress site without editing theme files.

== Description ==

RTR Custom Styles and Scripts is a lightweight plugin that allows WordPress site administrators to easily add custom CSS and JavaScript to their website without having to edit theme files. This is particularly useful for making small styling tweaks or adding tracking codes and other scripts.

= Features =

* Add custom CSS styles to your site
* Add custom JavaScript to your site's footer
* Add custom scripts to your site's header
* Add custom scripts to your site's footer
* Automatic code formatting for better readability
* CSS and JavaScript beautification
* Simple, user-friendly interface
* No coding knowledge required to use the plugin
* Changes persist even when you switch themes

= Use Cases =

* Add custom styling to your WordPress site without modifying theme files
* Insert Google Analytics, Facebook Pixel, or other tracking codes
* Add custom JavaScript functionality to your site
* Insert verification tags for various services
* Make minor CSS adjustments without creating a child theme

= Privacy =

This plugin does not collect any personal data. It simply provides an interface for site administrators to add custom code to their own website.

== Installation ==

1. Upload the `rtr-custom-styles-scripts` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'RTR Styles & Scripts' in your WordPress admin menu
4. Add your custom CSS and JavaScript as needed
5. Save your changes

== Frequently Asked Questions ==

= Will my custom code be lost if I switch themes? =

No, your custom code is stored in the WordPress database and will persist even if you change themes.

= Is this plugin compatible with caching plugins? =

Yes, this plugin is compatible with most caching plugins. However, you may need to clear your cache after making changes to see the effects.

= Can I add both CSS and JavaScript with this plugin? =

Yes, you can add both custom CSS and JavaScript. The plugin provides separate tabs for each type of code.

= Is the code added by this plugin minified? =

No, the code is not minified by default. Instead, it's automatically formatted for better readability with proper indentation and line breaks.

= Does the plugin format my code? =

Yes, the plugin automatically formats both CSS and JavaScript code for better readability. It adds proper indentation, line breaks, and spacing to make the code more organized and easier to maintain.

= Can I add code to specific pages only? =

The current version adds the code to all pages of your site. If you need page-specific code, you can add conditional logic within your custom JavaScript.
